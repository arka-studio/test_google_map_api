Tes API Key Google Map

Masukkan API key Google Maps kamu, lalu tekan tombol Tes. Key hanya dipakai sekali di browser dan tidak disimpan.

Belum dites

Isi API key, lalu klik tombol Tes untuk melihat hasil.

Yang dicek

Tool ini mencoba memuat Google Maps JavaScript API dari browser. Jika gagal, biasanya karena API belum aktif, billing belum aktif, key salah, atau domain belum diizinkan.
